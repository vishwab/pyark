GIT Notes =========

GIT Notes:
First clone the repo to a local branch, then make changes, then do "git status" to list the changes, then do "git add file-name-with-path" to add each file (or * for wild card), then do "git commit -m ", then do "git push", then create a pull request either by using the console URL or directly go to the git repo via browser and do it. Mention a reviewer. Once approved, do a Merge via browser.
